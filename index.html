SSSAJ [No Longer Available]


SSSAJ, pernounced sage, is a web application developed to allow users to send encrypted files back and forth while utilizing password protection on any encrypted files being stored on the database. The files are encrypted using the latest AES techniques, while only being able to be decrypted by a user with the proper pass phrase. The web applications was created for a class final project. The project was to create an application utilizing as much computer science related topics as possible while learning new techniques in the process. The group I was apart of used an agile methodology to achieve the finished product. We utilized our time management skills, broke up work individually, and worked on separate parts while piecing the whole application together. The team learned two different frameworks built upon Python for the project; Django and Flask. Django was used in the beginning, but the application was better suited for Flask in the end. Flask seemed to have more libraries available for use that would increase our chances of success. Learning how to use Jinja2 was a huge advantage over javascript code. Jinja can be embedded to produce dynamic quality functions in an instant without the security related issues.
